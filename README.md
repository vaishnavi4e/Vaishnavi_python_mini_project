## Project Overview
This project contains three Python programs demonstrating core Python concepts such as functions, lists, tuples, dictionaries, sets, loops, conditional statements, and mutable vs immutable objects.

## Files

### q1_result_analyzer.py
Student Result Analyzer
- Calculates total and average marks
- Assigns grades
- Identifies subjects with marks below 40

### q2_library_system.py
Library Book Management System
- Uses Dictionary, Tuple, List, and Set
- Supports adding books
- Borrowing and returning books
- Member registration

### q3_shopping_cart.py
Shopping Cart System
- Demonstrates mutable default argument issue
- Implements shopping cart functionality
- Calculates totals with discounts
- Shows tuple immutability

## Concepts Used
- Variables and Data Types
- Functions
- Lists
- Tuples
- Dictionaries
- Sets
- Loops
- Conditional Statements
- Mutable vs Immutable Objects

## How to Run

```bash
python q1_result_analyzer.py
python q2_library_system.py
python q3_shopping_cart.py
```
